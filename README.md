## Установка

1. **Склонируйте репозиторий:**
   ```sh
   https://github.com/echoisonline/auth-module.git
   cd auth-module
   ```
2. **Установите зависимости:**
   ```sh
   npm i
   ```
3. **Запутсите Dev сервер:**
   ```sh
   npm run start
   ```
4. **Откройте приложение в своём браузере:**
   ```
   http://localhost:5173
   ```
